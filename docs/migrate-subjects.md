# Commit d8f47e6 による変更への対応

✦ migrate-subjects.ts は、既存の講座（Course）に直接入力されていた「課目名」と「総時限数」を、新しく導入された「課目マスタ（Subject Manager）」へ自動的に移行するためのスクリプトです。

  主な役割
   1. デフォルト講座タイプの作成: 「General」という名前の講座タイプを1つ作成します。
   2. 既存講座の紐付け: すべての既存講座を、作成した「General」講座タイプに紐付けます。
   3. 課目マスタの生成: 既存の講座に登録されていたユニークな「課目名」をスキャンし、それらを「大課目（レベル1）」として課目マスタに登録します。
   4. リレーションの更新: 各講座の課目データを、新しく作成された課目マスタのID（subjectId）と紐付け直します。

  ---

  実行方法

  バックエンドのディレクトリに移動し、ts-node を使用して実行します。

   1 # バックエンドディレクトリへ移動
   2 cd backend
   3
   4 # スクリプトの実行
   5 npx ts-node src/scripts/migrate-subjects.ts

  注意事項
   * バックアップ推奨: データベースの内容を書き換えるため、実行前にDBのバックアップを取ることをお勧めします。
   * 一度きりの実行: 基本的に新環境への移行時に一度だけ実行することを想定しています。
   * 環境変数: 実行には .env ファイルに正しい DATABASE_URL が設定されている必要があります。
